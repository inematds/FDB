Gerência de Informações — três tipos principais

Classificação por forma de tratamento e uso dos dados, com foco em porquê e como utilizar.

1) Gerência de Informações Estruturadas

Exemplos: planilhas, bancos de dados relacionais, sistemas ERP.

  • Dados em formato fixo (colunas, linhas, campos).
  • SQL e SGBDs como MySQL, PostgreSQL, Supabase.
  • Ideais para grandes volumes de registros padronizados.
  • Exemplo prático: controle de clientes, vendas, estoque.

2) Gerência de Informações Não Estruturadas

Exemplos: textos, PDFs, imagens, áudios e vídeos.

  • Informações sem formato fixo exigem análise semântica.
  • Ferramentas de NLP e OCR transformam conteúdo em dados legíveis por IA.
  • Exemplo prático: analisar PDFs médicos, e-mails ou relatórios automáticos.

3) Gerência de Informações Vetoriais (RAG)

Exemplo: sistemas de busca inteligente com IA.

  • Uso de embeddings vetoriais para representar significado.
  • Perguntas em linguagem natural sobre documentos, textos, planilhas e PDFs.
  • Base de sistemas RAG (Retrieval‑Augmented Generation) — busca + geração com IA.
  • Exemplo prático: chatbot que responde com base em documentos internos.
